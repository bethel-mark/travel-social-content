# 小红书违禁词清单

> 本文件汇总小红书 / 抖音 / 微博等平台的常见违禁词与限流触发词。
> 所有生成的文案 **必须** 自查本清单，规避风险。

---

## ⚠️ 风险等级

| 等级 | 影响 | 处理 |
|---|---|---|
| 🔴 绝对禁止 | 直接限流 / 违规 / 封号 | 必改 |
| 🟡 高风险 | 大概率限流 / 降权 | 强烈建议改写 |
| 🟢 软性敏感 | 可能被审核提醒 | 建议优化 |

## 🟡 高风险（强烈建议改写）

### 1. 夸大宣传

- 「绝绝子」/「yyds」/「无敌」 → 「非常棒」「值得去」
- 「神仙地方」/「人间天堂」 → 「风景优美」「值得一去」
- 「太美了」反复出现 → 改为具体描述（蓝绿色湖面 / 金色山脊）

### 2. 诱导互动

- 「点赞过 X 万」/「不转不是中国人」 → 直接删除
- 「关注我下集更精彩」 → 「下期出 X 攻略」
- 「评论区扣 1」 → 可保留但避免强迫性话术

### 3. 价格 / 优惠虚假

- 「全网最低价」→ 删除或写具体来源
- 「限量 X 份」虚假 → 删除
- 「错过等一年」夸张 → 改为客观描述

### 4. 抄袭 / 搬运风险

- 整段引用其他博主文案 → 必须改写
- 未注明出处的图片 → 注明来源或用 AI Prompt 自创
- 抄袭品牌 Slogan → 删除或原创改写

## 🟢 软性敏感（建议优化）

### 1. 主观判断类（旅游常用）

- 「一定要去」 → 「值得一去」
- 「不去后悔」 → 「值得专程跑一趟」
- 「错过遗憾」 → 「有独特价值」

### 2. 比较类

- 「比 X 还美」 → 「有 X 的感觉，但更 Y」
- 「碾压 X 景点」 → 「值得和 X 一起规划」

### 3. 时效性夸大

- 「今年最火」 → 「近期热门」
- 「小红书都在推」 → 「搜索热度高」

## 🎯 旅游类专属词库

### 必改清单

| 违禁词 | 旅游场景合规替代 |
|---|---|
| 「必去」 | 值得一去 / 推荐 |
| 「绝了 / 绝绝子」 | 非常推荐 / 很棒 |
| 「yyds」 | 经典 / 难忘 |
| 「家人们谁懂啊」 | 姐妹们 / 朋友们 |
| 「一定要 X」 | 建议 X / 推荐 X |
| 「不来后悔」 | 错过可惜 |
| 「神级 / 神作」 | 高水准 / 经典 |
| 「出片」 | 拍照好看 / 出片率高 |

### 可保留清单（合规）

- ✅ 「绝美」（程度副词，相对安全）
- ✅ 「小众」（强调非热门，无夸大）
- ✅ 「宝藏」（网络流行语，相对安全）
- ✅ 「治愈」（情绪描述，无医疗指向）
- ✅ 「人少景美」（事实描述）
- ✅ 「小众秘境」（相对小众，无夸大）

### 时令类（注意时效）

- 「2026 最新」 → 「2026 春季版」
- 「限时 X 月」 → 具体日期范围

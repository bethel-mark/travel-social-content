# 文案公式库 · 5 种风格

> Skill 输出 Step 4「生成 5 种风格文案矩阵」的执行手册。每种风格都有 **钩子库 + 模板 + 实战示例**。

---

## A · 系列打卡型（最高实用价值）

**情绪定位**：实用、收藏、清单式  
**最佳平台**：小红书、B 站、YouTube  
**长度**：300-500 字 + 6/9 张图

### 钩子公式

```
📍<城市>
第①站·<景点A>  <一句话核心体验+emoji> 📸
·
📍<城市>
第②站·<景点B>  <一句话核心体验+emoji> 📸
·
...
#<城市旅游> #<景点A> #<景点B>
```

### 实战示例（乐山）

```
📍乐山
第①站·乐山大佛  71米弥勒佛，站在佛脚真的会被震撼到说不出话 📸
·
📍乐山
第②站·苏稽古镇  跷脚牛肉发源地，白墙黛瓦的清晨烟火气 📸
·
📍乐山
第③站·张公桥好吃街  本地人私藏夜市，9 块钱 30 个串串 😭
·
#乐山旅游 #乐山大佛 #跷脚牛肉 #四川美食
```

### 关键技巧
- 用 `·` 而非逗号，制造节奏
- 每个景点控制在 25 字以内
- 末景点必须含「食物」或「夜景」，形成闭环

## B · 情绪共鸣型（最易传播）

**情绪定位**：文青、治愈、心动  
**最佳平台**：朋友圈、Instagram、视频号  
**长度**：100-200 字 + 1 张大图

### 钩子公式

```
总要去趟<地点>吧
<看 A>，<抒情短句> 💙
<吹 B>的<风/雨>，<抒情> 🌬️
<走/听 C>，<身体部位>在<抽象名词>
<情绪金句> ·📍 <地点>
```

### 实战示例（稻城亚丁）

```
总要去趟稻城亚丁吧 🏔️
看牛奶海的蓝，是雪山流下的眼泪 💧
吹 6032m 处的风，呼吸都变得圣洁 🌬️
听洛绒牛场溪水，眼睛在天堂 ✨
一夜走过 318 国道无数道弯，值了
```

### 关键技巧
- 每句都对仗（动词 + 感受 + 体悟）
- 末尾必须有「**值**」「**该**」「**够**」等肯定词
- 不超过 5 行，避免抒情过度

## C · 反差吸引型（最高点击率）

**情绪定位**：悬念、反转、「被打脸」  
**最佳平台**：小红书、微博、抖音  
**长度**：150-250 字 + 6 张图

### 钩子公式

```
以为<刻板印象>
直到我看见了这些 👇
-
<A>，<反差金句>
<B>，<反差金句>
<C>，<反差金句>
-
<地点>，远比你想象的更<强情绪形容词>
```

### 实战示例（四川合集）

```
以为四川只有火锅
直到我看见了这些 👇
-
九寨沟的水，是上帝打翻的调色盘 🎨
稻城亚丁的雪，是地球最后的净土 ❄️
黄龙的钙化池，是人间瑶池的真实模样 ✨
四姑娘山的秋，是东方阿尔卑斯 🍁
若尔盖的草，是写给自由的情诗 🌾
-
四川，远比你想象的更「野」
```

### 关键技巧
- 第 1 句**必须**打破读者既有认知
- 每条子项 15 字以内，配 1 个 emoji
- 末句单独立项，重复核心形容词

## D · 数字攻略型（最强转化）

**情绪定位**：攻略、清单、避雷  
**最佳平台**：小红书、知乎、公众号  
**长度**：500-800 字 + 6 张图 + 实用卡片

### 钩子公式

```
<城市><天数><玩法>攻略｜人均<预算>
────────────────
✓ <必玩点 1>
✓ <必吃 2>
✓ <必拍 3>
⚠️ <避坑 1-2 条>
────────────────
#<城市旅游> #<数字>天<数字>夜
```

### 实战示例（宜宾）

```
宜宾 2 天 1 夜攻略｜人均 350 吃到撑
────────────────
✓ 蜀南竹海翡翠长廊 120km² 拍照太出片
✓ 李庄古镇白肉一条街，本地人 30 年老店
✓ 兴文石海溶洞探险，地心深处的星海
⚠️ 燃面只点 1 两就好，加面免费但会辣哭
⚠️ 五粮液酒厂需 1 周前预约
────────────────
#宜宾旅游 #蜀南竹海 #李庄古镇 #周末去哪儿
```

### 关键技巧
- 数字必须**精确**（不写「便宜」写「人均 350」）
- 末尾必含 **⚠ 避坑**（建立信任感）
- 可附「路线图截图」当配图

## E · 60 秒沉浸脚本（视频号 / 抖音）

**情绪定位**：视听、临场、BGM 驱动  
**最佳平台**：抖音、视频号、Reels  
**长度**：60 s 脚本 + 1 张封面图 + 1 段背景乐推荐

### 标准结构

```
[0-3s]   黑屏字幕 / 钩子金句
[3-10s]  高能镜头快剪（每镜 ≤ 2s）+ BGM 渐入
[10-30s] 中段沉浸（环境音 + 慢镜）
[30-50s] 实用干货（字幕条滚动）
[50-58s] 反差金句 / 抒情金句
[58-60s] 互动钩子 + 封面 CTA
```

### 实战示例（稻城亚丁）

```
[0-3s]   黑屏：「这辈子一定要去一次亚丁」
[3-10s]  快剪：云海冲上来 + 央迈勇倒影
[10-25s] 中段：洛绒牛场 + 牛奶海 + 冲古寺 慢镜
[25-30s] 日照金山爆发时刻，配 BGM《天地龙鳞》
[30-50s] 干货：路线 / 高反贴士 / 预算字幕条
[50-58s] 反差：「总以为亚丁是照片美，亲眼看到才懂什么叫幻灭」
[58-60s] CTA：「评论区告诉我，你最想去哪个」
```

### 推荐 BGM

| 情绪 | BGM 推荐 |
|---|---|
| 史诗 | 《天地龙鳞》王力宏 |
| 治愈 | 《稻香》周杰伦 / 《晴天》 |
| 自由 | 《平凡之路》朴树 |
| 中国风 | 《不染》毛不易 |
| 国潮 | 《踏山河》 |
| 民谣 | 《蓝莲花》许巍 / 《如果有来生》 |
| 异域 | 《Mongolian throat singing》|

### 关键技巧
- **前 3 秒**决定 80% 完播率，必须是钩子
- 字幕条在 0:30 后**才**出现（前面让人看画面）
- 结尾**永远**有互动问题

## 🎯 风格选择决策树

```
用户场景是？
  │
  ├─ 「我想收藏一份攻略」        → D · 数字攻略型
  ├─ 「我想发个朋友圈装小众」    → B · 情绪共鸣型  
  ├─ 「我想蹭热点/反常识」       → C · 反差吸引型
  ├─ 「我想一次性讲完几个点」    → A · 系列打卡型
  └─ 「我想做成视频号短片」      → E · 沉浸脚本型

默认组合 = A + B + C（覆盖 80% 场景）
```

## 🔁 风格融合技巧

一个目的地可以**叠加**使用多种风格：

```
# 主标题用 C（反差吸睛）
# 正文用 D（数字攻略）
# 配图用 B（情绪氛围）
# 视频号用 E（沉浸脚本）
```

这样一份方案可同时产出图文 + 视频。

## 📚 经典爆款句式参考库

### 开场钩子
- 「再不看 xx 攻略，就真来不及了」
- 「作为一个旅游博主，我被问得最多的是 xx」
- 「比 xx 更值得去 10 倍的，是这个...」
- 「如果 xx 是真的，那 yy 一定存在」

### 转折钩子
- 「以为 / 结果」
- 「都说 / 其实」
- 「本地人 / 才知道」

### 情绪金句
- 「人间值得」「眼睛在天堂」
- 「值回所有 318 国道的弯」
- 「才知道什么叫 '人间瑶池'」

### 互动钩子
- 「你想去哪个？评论告诉我」
- 「收藏这篇，下次出行直接抄作业」
- 「建议转给那个总说『没时间去』的朋友」

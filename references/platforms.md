# 平台格式规范

> 同一份内容方案需要在 4 个主流平台分发，每个平台的字数、标签、排版、互动话术都有自己的「脾气」。本文档是 Skill 输出 Step 6「平台适配」的执行手册。

---

## 📱 1. 小红书（RedNote / Xiaohongshu）

### 卡片信息

| 维度 | 规范 |
|---|---|
| 标题字数 | **12-18 字**（最长 20）|
| 正文长度 | **300-600 字** 为佳（最长 1000）|
| 配图张数 | **6 或 9 张**（同色系拼接最佳）|
| 视频长度 | 60-90 s 沉浸式 |
| 标签数量 | **15-20 个**（精准 + 泛流量分层）|

### 标题模板（钩子库）

| 类型 | 句式 | 示例 |
|---|---|---|
| **数字型** | `四川8大秘境+语气词+人群` | 「四川 8 大秘境，去过 3 个算你厉害」 |
| **反差型** | `以为X / 结果 Y` | 「以为四川只有火锅，直到我去了稻城」 |
| **悬念型** | `再不看就... / 最后.....` | 「再不看这版九寨攻略，就真错过了」 |
| **共鸣型** | `金句+情绪词` | 「在川西的草原上做一场自由的梦」 |
| **盘点型** | `城市+玩法+预算` | 「成都 3 日游，人均 1500」 |

### 正文必含模块

1. **钩子第一句**（决定 3 秒停留率）
2. **子景点清单**（每个 2-3 行，配 1 张图）
3. **实用信息**（门票/季节/交通）
4. **互动钩子**（结尾提问）

### 标签分层模板

```
#大词（流量）      3-5 个  #四川旅游 #川西自驾 #旅行打卡
#目的地（精准）    5-8 个  #九寨沟 #稻城亚丁 #四姑娘山
#场景（人群）      3-5 个  #小众旅行地 #秋天去哪儿玩 #5 天 4 晚
#情绪（蹭热搜）    2-3 个  #治愈心灵 #蓝色星球
```

### 发布时间
- **最佳**：周二 / 周四 18:00-20:30
- **次佳**：周末 10:00-12:00
- **节假日**：提前 1-2 周发布收藏型攻略效果最佳

### 互动话术套件
- 「你最想去哪个？评论区留下 ❤️」
- 「收藏这篇，下次出行直接抄作业！」
- 「建议转给那个总说『没时间去』的朋友」

## 📷 2. Instagram（Reels + Carousel）

### 卡片信息

| 维度 | 规范 |
|---|---|
| 标题（Caption） | **50-150 字**为佳（最多 2200） |
| Hashtag | **20-30 个**（IG 算法偏爱大量） |
| Reel 长度 | 15-90 s（**30 s 内完成率最高**） |
| Carousel 张数 | **3-10 张**（5 张完播率最佳） |
| 视频比例 | 9:16（Reel） / 1:1（Feed） |

### Carousel 第 1 张的「黄金封皮」

- 大字标题（人像 + 地点）
- 对比强烈的色块
- 「Save this for…」提示收藏
- 一句话钩子（英文短句）

### Reels Hook 模板

```
"Stop everything. If you haven't added <place> to your travel list,
this is your sign."

📍 <place> — '<one-line tagline>'

Save this. Trust me.
```

### Hashtag 配方

```
#核心地点（5）  #china #travelchina #sichuan #jiuzhaigou #daocheng
#类别（5）      #nature #mountains #travelphotography #wanderlust #adventure
#情绪（5）      #mindfulness #solotravel #couplegoals #bucketlist #heaven
#UGC 标签（3）  #natgeo #lonelyplanet #cntraveler 蹭大号流量
#平台独占（3）  #reels #reelsinstagram #explore
```

### 发布时间（目标时区）
- **美东**：07:00-09:00 EST
- **欧西**：12:00-14:00 CET
- **东南亚**：11:00-13:00 SGT
- **中东**：10:00-12:00 GST

## 🟢 3. 朋友圈（WeChat Moments）

### 卡片信息

| 维度 | 规范 |
|---|---|
| 文字长度 | **1-3 行**（最多 6 行，超过会折叠） |
| 配图张数 | **1 张**（单图爆款）或 **6/9 张**九宫格 |
| 视频长度 | 10-30 s |
| 互动机制 | 点赞 + 评论，无 hashtag |

### 单图模式（爆款率最高）

```
<强情绪一句话>
   ·
<地点 + 时间>
```

**示例**：
```
在稻城亚丁的第三天，遇到了日照金山。
   ·
   · 6,032m · 2026.10
```

### 九宫格模式

- 9 张图按 **空间顺序** 或 **时间顺序** 排列
- 第 1 张必须是**封面级**（最具视觉冲击力）
- 配文 1-2 行情绪 + 隐藏细节（emoji 当分隔符）
- 末张可以是路线/价格截图，引导私聊

### 互动爆点
- 「猜猜这是哪？」配 1 张无文字线索图
- 「明早 9 点删」制造紧迫感
- 「点过赞的都是将要来」筛选真朋友

## 🔵 4. 微博（Weibo）

### 卡片信息

| 维度 | 规范 |
|---|---|
| 文字长度 | **140 字以内**为佳（最多 2000，但超过折叠） |
| Hashtag | **5-10 个**（不是越多越好） |
| 配图张数 | **9 张**为佳 |
| 视频长度 | 60 s 内 |
| 重要规则 | 涉及明星/政治敏感词必须规避 |

### 微博钩子模板

```
#<大词># <强情绪一句话>
📍<地点>
🔗<链接或话题>
```

### 蹭热搜策略
- 在「中国国家地理」「视觉中国」等蓝 V 下评论高质量攻略
- 用「#带着微博去旅行#」「#最美旅行照#」等平台级话题
- 与本地蓝 V 互动（@乐山发布 @九寨沟管理局）

## 🎯 平台适配决策树

```
用户想发什么？
  │
  ├─ 攻略 / 干货      → 小红书（图文长 + 标签多）
  ├─ 视觉奇景 / 出国  → Instagram（Reel + Carousel）
  ├─ 日常情绪 / 朋友  → 朋友圈（短 + 单图）
  └─ 国内社交 / 蹭流量 → 微博（话题驱动）
```

**默认输出**：小红书为主版本 + 朋友圈精简版（覆盖 80% 用户需求）。
**完整输出**：请求加 `全平台` 时，输出 4 个版本。

---

## ⚠️ 通用合规清单

发布前必查：

- [ ] **票价 / 时间**：超过 6 个月的官方数据需标注「以官方为准」
- [ ] **AI 生成图**：建议加 `#AI生成` 标签，遵守《人工智能生成合成内容标识办法》
- [ ] **人物出镜**：取得肖像权授权，避免未成年人正面特写
- [ ] **风光版权**：标注参考图出处，避免盗用摄影师作品
- [ ] **平台规则**：不刷量、不诱导关注、不恶意引流
- [ ] **宗教 / 民族**：尊重少数民族信仰，不消费宗教符号
